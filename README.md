## 🧸 历史

> ~~*（其实就是一些来自原作者的废话）*~~  

这个项目的起源很简单：我的一位朋友找我帮忙，想做一个小工具来满足他的小需求。
我本人其实并不玩 MAIMAI，但在他演示了获取二维码的操作后，我理所当然地觉得通过自动点击的方式就可以实现他的想法，
于是就做了这样一个简单的小工具。
后来，他把这个项目分享到了哔哩哔哩给更多人使用。随着用户的增加，大家对工具提出了更多问题和需求。
为了更好地满足大家的期待，我抽时间对工具进行了重构，并决定将这个项目开源，让更多人能一起使用和改进它！🥺
